PAINEL DE DEVEDORES (CONTROLE DE EMPRÉSTIMOS)
=================================================

Instruções de Uso:
1. Abra o arquivo 'index.html' no seu navegador.
2. O sistema iniciará com um banco de dados local (no seu navegador).
3. Use o botão "Novo Empréstimo" para cadastrar um devedor.
4. Para dar baixa em um pagamento:
   - Clique no ícone de detalhes (olho) na tabela.
   - Clique sobre a parcela desejada para mudar de "Pendente" para "Pago" (e vice-versa).
5. Seus dados ficam salvos automaticamente (LocalStorage).

Funcionalidades:
- Cadastro de Devedores (Nome, Valor, Parcelas, Data).
- Controle financeiro (Total Emprestado, Recebido, Pendente).
- Filtros por Nome, Tipo e Status.
- Exportação para Excel (.csv).
- Design Premium Inter Neon Pro.

Notas:
- Os dados são salvos apenas no navegador que você está usando. Se limpar o cache, os dados somem.
- Funciona 100% Offline.
